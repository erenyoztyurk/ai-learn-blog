# Giriş

Merhaba! Bu yazıda **Next.js** ve **MDX** kullanarak nasıl modern bir blog altyapısı kurabileceğinizi anlatacağım.

## Neden MDX?

MDX, Markdown içinde JSX bileşenleri kullanmanıza olanak tanır. Bu sayede:

- 📝 Normal markdown syntax'ı kullanabilirsiniz
- ⚛️ React bileşenlerini doğrudan yazılarınıza entegre edebilirsiniz
- 🎨 Daha interaktif ve zengin içerikler oluşturabilirsiniz

## Örnek Kod Bloğu

```javascript
// next.config.js
const withMDX = require('@next/mdx')({
  extension: /\.mdx?$/,
});

module.exports = withMDX({
  pageExtensions: ['js', 'jsx', 'mdx'],
});
